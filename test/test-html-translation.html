To enable IPVS mode, you need to modify the kube-proxy configuration:

apiVersion: kubeproxy.config.k8s.io/v1alpha1
kind: KubeProxyConfiguration
mode: "ipvs"

This configuration tells kube-proxy to use the IPVS load balancer instead of the default iptables implementation.
